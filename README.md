# voronoi-reactive-shader

Reactive layered Voronoi — 3-layer multi-scale Voronoi with animated seeds, edge
glow, spatial warp, and HSV coloring. Designed as a control surface for external
audio reactivity.

**[Interactive math walkthrough](https://wday.github.io/voronoi-reactive-shader/)**
— the same theory content as `THEORY.md`, with embedded WebGL demos.

## Structure

- `shaders/` — ISF fragment shaders
- `preview/` — browser preview harness
- `ffgl-rs/` — FFGL plugin build pipeline (Rust)
- `docs/` — interactive theory page (served by GitHub Pages)
- `scripts/` — deploy, preview, and session docs
